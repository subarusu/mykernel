This 3.18.x kernel tree is based on Q branch of CAF source.

## Cloning
`git clone https://github.com/shivamjadon/Drona_Kernel_Rosy.git -b q-caf --single-branch`

## Building
Please build using:

`drona-rosy_defconfig`
